# 桌面灵感挂件

这里存放“桌面灵感挂件”的需求、设计决策和后续实现资料。

## 当前目标

做一个始终在桌面上、能让你快速记下想做事项的小挂件，避免灵感在稍后被遗忘。

## 需求澄清状态

正在确认核心使用方式；尚未开始实现，以免先做出不符合你习惯的交互。

## 已知需求

- 使用场景：每天随时出现很多点子和待办。
- 核心问题：如果不能立刻记录，点子很快会忘记。
- 期待形态：桌面小挂件。
- 产品形态：独立的本地桌面软件；首版离线可用，后续可打包分发并扩展授权与付费能力。
- 首版平台：Windows。
- 首版条目：每条只保存一句话，不含分类、优先级、截止时间或提醒。
- 层级：一条主 Note 可包含多个子 Note，用于把一个想法拆成下一步；子 Note 与主 Note 都是简短的一句话。层级仅一层，不支持子 Note 继续嵌套。
- 数据范围：首版只在本机离线保存，不提供登录、云同步或移动端；未来可新增 Android 与 Windows 同步。
- 列表展示：挂件显示最近 5–10 条灵感条目，并提供进入完整列表的入口。
- 挂件范围：主挂件只显示未完成的主 Note；已完成的主 Note 与子 Note 仅在完整列表的已完成视图显示。
- 完成状态：条目可标记为“已完成”，并保留在完整列表的已完成视图中。
- 父子完成规则：主 Note 有子 Note 时，必须等所有子 Note 完成后才自动标记为已完成；仍有未完成子 Note 时不可直接完成主 Note。
- 删除：条目可删除；删除前必须二次确认。
- 编辑：主 Note 和子 Note 都支持原地编辑；Enter 保存，Esc 取消。
- 搜索：完整列表支持关键词搜索；默认仅显示一个尺寸克制的搜索图标按钮，点击后才展开搜索框。主挂件不显示搜索框。
- 时间：完整列表显示每条 Note 的创建时间；主挂件不显示时间。
- 可见性：默认常驻桌面，窗口可自由拖动、缩放和关闭；关闭后必须可随时重新打开。
- 重新打开：提供系统托盘菜单，并以 `Ctrl + Alt + Space` 作为显示/隐藏挂件的全局快捷键。
- 视觉与布局：采用现代半透明卡片；顶部快速输入，中部展示可展开的主 Note / 子 Note 及子项完成进度，底部进入完整记录列表。
- 置顶：提供始终置顶开关，默认关闭；用户选择将持久化。
- 启动策略：不提供开机自动启动；软件只在用户手动打开后运行。
- 首版交付：生成可安装的 Windows 安装包。
- 实现边界：使用 Electron 桌面应用；界面只通过本地 Note 仓库读写数据，未来同步功能以此为扩展点。
